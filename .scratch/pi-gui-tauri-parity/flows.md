# pi-gui beta.32 Desktop Flow Matrix

## Frozen Reference

- Repository: `../pi-gui`
- Commit: `7bed8a9ed969905cf0f4dc1c5eaa84d2ce1ab841`
- Product target: behavioral parity in Tauri, not Electron or pixel parity.
- Rule: every included row has exactly one owning ticket. Ticket 43 rejects missing, duplicate, or unverified ownership.

## Locked Architecture

- Tauri 2 and Rust own native capabilities, durable app state, recovery, and window projections.
- React JavaScript, Radix, and Tailwind own presentation only.
- Bun manages dependencies and builds.
- Bundled Node 20 runs `@mariozechner/pi-coding-agent@0.73.1`.
- The Rust/host protocol is versioned, validated, request-addressed, streaming, cancellable, and crash-safe.
- Upstream Pi owns sessions, auth, providers, models, skills, and extensions.
- Pi CLI and desktop coexist through external refresh and single-writer leases.

## Explicit Exclusions

- Update checking and automatic updates.
- Website, video/demo production, and Homebrew automation.
- Signing, notarization, DMG distribution, Linux, and Windows.
- Standalone computer-use.
- Migration of app-owned `pi-gui` state.
