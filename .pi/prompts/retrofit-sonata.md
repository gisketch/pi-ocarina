# retrofit-sonata

Use when adapting an existing project into this harness.

Rules:

- Inventory source, docs, commands, agent files, and architecture.
- Preserve useful project knowledge by moving, linking, or summarizing into `docs/`.
- Keep root instructions short.
- Do not change app behavior unless requested.
- Add checks to `docs/quality.md` only after verifying or clearly marking them unverified.
- Run `./scripts/check-sonata.sh`.
