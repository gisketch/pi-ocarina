# UI Increment Map

## Visible Now

- Unified workspace/thread sidebar.
- Conversation transcript.
- Composer and its model, thinking, attachment, send, steer, and stop controls.
- Runtime errors and prompts required to complete an active chat flow.
- Collapsible, resizable Changes workbench with shared editor diff and compact file tree.
- Dedicated Settings workspace with General, Appearance, Providers, and Notifications; entry is pinned to the main sidebar footer.

## Implemented But Hidden

- New-window action and model onboarding guidance.
- Integrated terminal.
- Resources, session tree, and fork navigation entry points.
- Extension dock and queue editor.
- Advanced workspace add, rename, remove, worktree, and Finder actions.
- Thread pin, reorder, archive, rename, and archived-thread controls remain implemented and should be consolidated into contextual menus later.

These features are not deleted. Restore them incrementally through deliberate Codex-like dialogs, popovers, or contextual toolbar actions instead of adding another permanent column.
